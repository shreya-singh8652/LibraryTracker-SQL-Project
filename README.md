# Library Tracker Database System

## Project Overview

Library Tracker is a MySQL database project designed to manage books, members, and borrowing records in a library. The project demonstrates database design, data management, reporting, and advanced SQL concepts.

## Database Tables

* Books
* Members
* Borrow_Records

## Features

* Store and manage book records
* Maintain member information
* Track borrowed and returned books
* Generate borrowing statistics
* Identify currently borrowed books
* Find the most borrowed book

## SQL Concepts Used

* CREATE DATABASE
* CREATE TABLE
* PRIMARY KEY
* FOREIGN KEY
* INSERT
* SELECT
* JOIN
* LEFT JOIN
* GROUP BY
* COUNT()
* ORDER BY
* LIMIT
* DELETE
* VIEW
* STORED PROCEDURE

## Sample Queries

* Find books borrowed by a specific member
* Display overdue borrowed books
* Count books borrowed by each member
* Identify the most borrowed book
* View currently borrowed books
